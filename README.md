Super Store Sales Dashboard

Overview
The Super Store Sales Dashboard is a Power BI project developed to analyze and visualize sales performance data effectively. This dashboard helps in understanding key business metrics such as sales, profit, customer segments, regional performance, and product trends through interactive visualizations and reports.
The project is designed to support data-driven decision-making by providing clear insights into business operations and sales growth patterns.

Features
Interactive Power BI Dashboard
Sales and Profit Analysis
Regional and Category-wise Performance
Customer Segment Insights
Order and Quantity Analysis
KPI Cards for Quick Overview
Dynamic Filters and Slicers
Data Visualization using Charts and Graphs

Technologies Used
Power BI
Data Visualization
Data Analysis
DAX
Excel / CSV Dataset

Dashboard Insights
The dashboard provides insights such as:
Total Sales and Profit
Best Performing Categories
Region-wise Sales Distribution
Customer Segment Contribution
